# Dmoney API Testing Project

## Summary:
In this demo FinTech system, we can create **Admin**, **Agent**, **Customer**, and **Merchant** roles.  
They can send, receive, deposit, withdraw, and make payments just like a real digital wallet.  
The **Admin** manages all user creation and fund distribution using the Dmoney REST API and the Dmoney Portal web interface.  
This platform is mainly used for **API testing and QA training**.



### Technical Scope:
- node.js  
- postman  
    

### How to run this project:

1. Import the Dmoney API collection into **Postman**.  
2. Set the following variable in your Postman environment:  
baseUrl = https://dmoney.roadtocareer.net
3. Execute the collection from top to bottom:  
**Admin → Agent → Customer → Merchant.**
4. Verify that each step returns the correct JSON response and status codes (`200`, `201`, `400`, `401`, `404`).
